# 🎵 Music Player Web App

A modern and interactive music player inspired by Spotify UI.  
Built using HTML, CSS, and JavaScript with smooth controls and dynamic UI.

---

## 🚀 Live Demo
👉 https://my-musicplayer-20.netlify.app

---

## 📸 Features

- 🎧 Play / Pause music
- ⏭ Next & Previous track
- 🎚 Progress bar control
- 🎵 Dynamic song list
- 💻 Responsive UI
- ✨ Clean and modern design

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
